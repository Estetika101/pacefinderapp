Loader 1 of 5

Rev Counter Sweep
Use for short loads (< 1 s). Sessions home / track / game pages, lap-times table, headline strip on Deep Dive.

Loader 2 of 5

Christmas Tree (Starting Lights)
Use for medium loads (1–3 s). Deep Dive tab first switch, Telemetry tab first switch. Lights drop red one-by-one then all flick green.

Loader 3 of 5

Pit-stop Tyre Change
Use for long loads (≥ 3 s). AI Coaching analysis. Each tyre fills clockwise FL → FR → RR → RL, then flashes green.

Loader 4 of 5

Lap Counter Tick
L 123456789
Use as in-row skeleton placeholder. Lap-times table while waiting for /sessions/session/data, per-lap rows in events list.

Loader 5 of 5

Tyre Warming
Ambient page-level loader. Session detail initial load. Each corner cycles cold → optimal → hot, slightly out of phase, ripples through the grid.

Try System Settings → Accessibility → Reduce Motion (macOS) or your OS equivalent — every loader collapses to a static barber-pole bar.
