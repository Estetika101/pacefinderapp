# Code of Conduct

Pacefinder follows the **Contributor Covenant v2.1**.

The full canonical text is at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short:

- Be respectful and constructive in issues, PRs, and discussions.
- Welcome new contributors and questions, even basic ones.
- Disagree about ideas, not people.
- Don't post content that targets, demeans, or excludes others.

## Reporting

If you see behavior that violates this code, contact the maintainer via
GitHub Security Advisories on this repo, or email the address listed on
the maintainer's GitHub profile.

Reports are handled privately and the reporter's identity is not shared
without consent. The maintainer may warn, restrict, or ban anyone whose
behavior is judged inconsistent with this code.

## Scope

Applies to all project spaces — issues, pull requests, discussions, the
project README, and any space representing the project in public.
